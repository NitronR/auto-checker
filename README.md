# auto-checker

Prerequisite: Setup node v18.16.0

## Project setup
```
npm install
```

Add OPENAI_API_KEY in src/utils/ai_utills.js

### Compiles and hot-reloads for development
```
npm run serve
```

### Usage
Columns expected in question paper CSV file:
1. Question
2. Max Score
3. Expected Answer

Columns expected in anser sheet CSV file:
1. Question
2. Answer

### Compiles and minifies for production
```
npm run build
```
